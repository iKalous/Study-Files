SSH-key 与身份校验

SSH-key 生成
ssh-keygen -t rsa -C "邮箱地址"
cd ~/.ssh
cat id_rsa.pub （显示公钥）

这样就会生成 id_rsa（私钥） 和 id_rsa.pub（公钥上传至github）

关联SSH-key到GitHub账户

点击头像然后进入setting，可以点击ssh & gpg keys
将显示的公钥输入即可

记得检验密钥是否设置成功！
输入 ssh -T git@github.com

接下来回到主页创建新的仓库（第一次显示的是new repository，之后就是new）
该填的备注就填一下，但是有一些详细的设置不是很清楚（课上没讲）

接下来是github网站的用法

git 与 github 交互常用命令

克隆git仓库到本地，让自己能查看或修改项目 git clone + 该仓库的ssh key

拉取远程仓库数据并合并到当前分支 git pull （本质上是两个操作 git fetch 和 git merge）

推送本地仓库的commit记录到 git push A B（这里的A一般是git remote -v中显示的origin B是当前你要上传的分支，一般是main）
